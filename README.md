# RulesZenCI  Issue Tracker
Welcome to the [RulesZenCI Issue Tracker](https://github.com/Zen-CI/rules.zen.ci/issues).

This repository contains no code. It exists specifically for reporting and
working on issues for RulesZenCI. Please use this tracker for:
* filing bug reports
* requesting new features

Please [search the existing issues](https://github.com/Zen-CI/rules.zen.ci/search?state=open&type=Issues) before filing new ones.

# Security issues

If you have discovered a security issue with RulesZenCI, please contact the ZenCI Security Team directly at [security@zen.ci](mailto:security@zen.ci). We manage security issues separately in a private repository until the issue has been resolved. Even if you're not sure if it's a security problem, please contact the security team before filing an issue.
